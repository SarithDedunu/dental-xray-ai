# Dental X-Ray Diagnosis System (Frontend)

This project is the frontend part of an AI-powered dental X-ray diagnosis system. It offers a clean and intuitive UI for users to upload dental X-ray images and receive AI-driven diagnostic results with visual explanations.

## Getting Started

To run this project locally, follow these steps:

1. Clone the repository:
   git clone <repository-url>
   cd <repository-folder>
   
2. Install all required dependencies:
   npm install

3. Start the development server:
   npm start
